# node-wechat-dev
从零开始构建一个基于Node.js和Koa框架的微信公众号后台

## Content
- 01、 环境配置 [博客地址](https://blog.csdn.net/MiMicoa/article/details/83715464)

## Requirements
- Node v8.7.0+
- Npm v5.4.2+
- TypeScript v2.5.2+

## Quick Start
```bash
$ git clone git@github.com:coolboygym/node-wechat-dev.git
$ cd node-wechat-dev
$ git checkout stage1
$ npm install
$ tsc
$ node app.js
```
